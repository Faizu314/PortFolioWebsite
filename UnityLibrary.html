My Unity Library

Overview

This is a library of custom unity packages that I am building as I work on any Unity project. Whenever I design any system that can be used anywhere else, or even small utility functions, I add it to this repository. This way I can reuse code and when enough time passes I will be very sure that the code I am using is reliable.

Repository

I have uploaded this library as a public open source repository on github, Click here. I regularly maintain and test the code.
